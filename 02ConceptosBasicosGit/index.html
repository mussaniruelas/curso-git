Introducción

En esta sección, aprenderás los conceptos básicos de Git y por qué es una herramienta esencial para el control de versiones. Git te permite gestionar cambios en el código de manera eficiente y colaborativa, asegurando la integridad y el seguimiento de cada modificación en los proyectos.

Instalación

Guía paso a paso para instalar Git en diferentes sistemas operativos. Asegúrate de tener permisos de administrador y sigue los comandos específicos para tu entorno (Windows, macOS o Linux).

  • Windows: Descargar el instalador desde el sitio oficial y seguir el asistente de instalación.
  • macOS: Instalar usando brew install git si tienes Homebrew.
  • Linux: Usar el comando sudo apt-get install git en distribuciones basadas en Debian.

Comandos Básicos

Aquí encontrarás una lista de comandos esenciales para trabajar con Git:

Inicializar Repositorio

git init

Crea un nuevo repositorio en el directorio actual.

Agregar Archivos

git add .

Añade todos los archivos al área de preparación.

Realizar un Commit

git commit -m "mensaje"

Guarda los cambios en el historial del repositorio.

Ver Estado

git status

Muestra el estado actual del repositorio.
